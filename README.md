# JWT Authentication System

Full Stack Authentication System using React and Spring Boot.

## Tech Stack
Frontend:
- React (Vite)
- React Router
- Axios

Backend:
- Spring Boot
- Spring Security
- JWT Authentication

Database:
- MySQL

## Features
- User Registration
- User Login
- JWT Token Authentication
- Protected Dashboard
- Logout Functionality

## Project Architecture

React Frontend
      ↓
Spring Boot REST API
      ↓
JWT Authentication
      ↓
MySQL Database

## Run Backend

Run Spring Boot application.

Backend runs on:
http://localhost:8080

## Run Frontend

Go to frontend folder and run:

npm install  
npm run dev

Frontend runs on:
http://localhost:5174
